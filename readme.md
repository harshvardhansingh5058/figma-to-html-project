# Figma to HTML/CSS Website

## 📌 Overview
This project is a static website created by converting a Figma design into HTML and CSS. The focus was on accurate UI implementation, clean layout structure, and proper alignment.

---

## 🧩 Features
- Header with logo and contact details
- Navigation menu
- Sidebar with links and news section
- Enquiry form with proper alignment
- Footer with useful links and contact info

---

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox)

## 📱 Responsiveness
- Flexbox used for layout
- Basic responsive adjustments
- Proper alignment using fixed label width

---

## 🚧 Challenges Faced
- Form alignment issue (fixed using flexbox)
- Matching spacing with Figma design
- Handling layout overflow

---

## ❌ Limitations

- Limited mobile responsiveness
